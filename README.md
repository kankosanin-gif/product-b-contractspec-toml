# Product B — ContractSpec TOML

A TOML-first contract definition standard for agent task execution.

## Why TOML
- Better readability for long-lived ops contracts
- Git diff friendly for reviews
- Explicit sections for governance and verification

## Included
- `CONTRACTSPEC_TOML_EXAMPLE.toml`
- `CONTRACTSPEC_TOML_GUIDE.md`

## Intended Use
1. Attach a contract file to every implementation issue
2. Validate required sections before execution
3. Require evidence links before gate transitions
